# Image Anomaly Detection – Pole, Fire & Wire Monitoring 🔍🔥⚡

A computer vision project that detects anomalies in images using a combination of
**CNN-based anomaly detection** and **YOLO object detection**, built fully from scratch
using **PyTorch and OpenCV** (used third-party platforms like Roboflow).

This system is designed for **infrastructure and plant monitoring use cases** such as:
- Fire detection
- Abnormal object presence
- Pole and wire anomaly detection

---

## 🚀 Key Features

- Image preprocessing using OpenCV
- CNN-based anomaly detection model
- YOLO-based object detection
- Combined decision logic for anomaly classification
- Modular and scalable project structure
- Fully version-controlled using Git & GitHub

---

## 🧠 Project Architecture
Input Image
│
├──► OpenCV Preprocessing
│
├──► CNN Anomaly Detection
│
├──► YOLO Object Detection
│
└──► Final Anomaly Decision

## 🛠️ Tech Stack

- **Programming Language:** Python 3
- **Deep Learning:** PyTorch
- **Computer Vision:** OpenCV
- **Object Detection:** YOLO
- **Utilities:** NumPy, Matplotlib
- **Version Control:** Git & GitHub
